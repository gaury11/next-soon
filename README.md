# 목적

next.js 를 사용하여 웹 화면을 생성한다.

# 순서

1. 프로젝트 생성
2. ESlint, Prettier 설정
3. 화면 개발
4. 배포

## 1. 프로젝트 생성

```
// 프로젝트 생성
npx create-next-app next-soon

cd next-soon

npm run dev
```
